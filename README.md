# Luckydye Shade

## How do I install these formulae?

`brew install luckydye/shade/<formula>`

Or `brew tap luckydye/shade` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "luckydye/shade"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
